**SkillSphere** is an online learning platform designed to provide a seamless and interactive learning experience for users. It helps learners access structured educational content in a simple and modern interface.

---

## 🌐 Live Demo

👉 https://category-a8-orange.vercel.app/

---

## 🚀 Purpose

The purpose of **SkillSphere** is to create a centralized online learning environment where users can explore courses, enhance their skills, and track their learning progress efficiently.

---

## ✨ Key Features

- 📚 User-friendly online learning interface
- 🔍 Easy navigation through courses and categories
- ⚡ Fast and responsive UI
- 🎯 Focused learning experience
- 🔐 Secure authentication system
- 💡 Modern design with reusable components

---

## 📦 NPM Packages Used

- **Hero UI**
- **React Icons**
